# 3D Models App
### This is the second assignment for e-commerce course, which is done in team.
## Installation and Running Process
1- Install Wampoon here https://wampoon-box.github.io/getting-started/
For more information visit https://wampoon-box.github.io/
2- Run the exe file 
- Start the http and mariadb server.
- Click on localhost and phpMyAdmin to see the database and the app on browser on two different pages. 
<img width="500" height="500" alt="image" src="https://github.com/user-attachments/assets/42c7040b-6d37-4f0a-8b2c-6b2a335b445a" />

3- Clone the repository in htdocs folder under Wampoon.
4- Reload the localhost browser you will now see the app. Upon clikcing on the app you will be able to use it.
