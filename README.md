# 3D Models App

### Overview
This is the second assignment for the e-commerce course, completed as a team project.

## Installation and Running Process

1. **Install Wampoon**  
   Download and install Wampoon from:  
   [Getting Started with Wampoon](https://wampoon-box.github.io/getting-started/)  
   For more information, visit: [Wampoon Documentation](https://wampoon-box.github.io/)

2. **Run Wampoon**  
   - Start the **HTTP** and **MariaDB** servers.  
   - Open **localhost** and **phpMyAdmin** to view the database and the app in two separate browser pages.  
     
   ![Wampoon Setup](https://github.com/user-attachments/assets/42c7040b-6d37-4f0a-8b2c-6b2a335b445a){ width=500 height=500 }

3. **Clone the Repository**  
   Clone this project into the `htdocs` folder under Wampoon.

4. **Launch the App**  
   - Reload the **localhost** in your browser.  
   - Click on the app link to start using it.
5. **files: ERD, UserGuide, and README are in project root folder and reference.md is Implementation Resources in project board**

6. for bootstrap.php issues run `composer update --ignore-platform-req=php`
